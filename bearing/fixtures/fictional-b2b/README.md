# RelayDesk

RelayDesk is a fictional B2B SaaS product that imports approved customer
records into a shared onboarding queue. This tiny repository is deterministic,
credential-free, and intended only for public demonstrations.

The product facts live in `docs/product-facts.json`. A deliberately unsupported
claim lives in `docs/launch-draft.json` so an independent review can find and
correct it. All names and figures are invented.
